`/dubbing` と `/srt` スキルに関するよくある質問です。セットアップと使い方については [README](README.md) をご覧ください。

### 利用するには何が必要ですか？

Node.js 18以上と Perso Dubbing APIキーです。スキルをインストールしたら、「この動画を吹き替えて」と言うだけです。→ [APIキーを取得](https://developers.perso.ai/api-keys)

### APIキーはどのように登録しますか？

初回実行時にキーファイルが自動的に開きます — そのファイルに**APIキーのみ**を貼り付けて保存してください（暗号化された後、ファイルは削除されます）。**キーをチャットに貼り付けないでください。** 手動確認: `npm run key:check`

### 料金はかかりますか？

スキルのコード自体は無料（MIT）ですが、吹き替え処理はPerso APIを通じて実行され、クレジットが課金されます: 吹き替え ≈ 1クレジット/秒、リップシンク ≈ ×2、音声分離 ≈ ×0.5。4Kソースは pro/business/enterprise プランで ×3 課金されます。課金額はサーバー側の計算が正となります。

### 何を入力できますか？

ローカルファイル、フォルダ全体（バッチ処理）、またはURL（YouTube、TikTok、Google Drive、Vimeoを含む）です。サイズが大きすぎる動画や非常に長い動画は自動的に分割・処理され、最後に結合されます。

### 複数言語への吹き替えや、複数ファイルの一括処理はできますか？

はい。1つのコマンドに複数言語を指定できます（`--target en,ja,zh`）— ソースのアップロードと分割は一度だけ行われ、言語ごとに再利用されます。複数のファイル・フォルダ・URLを1回の実行で混在させることもできます。

### 結果はどこに保存されますか？

デフォルトでは元の動画と同じ場所に保存されます。`--out` で指定したフォルダに保存することもできます。すべての実行結果は、Perso ポータル（<https://portal.perso.ai>）上のプロジェクトとしても保存され、そこから再ダウンロードしたり、他の形式を取得したりできます。

### リップシンクとは何ですか？

口の動きを吹き替え音声に同期させる機能です。吹き替えの後に実行され、動画にのみ対応し、処理時間はかなり長くなり、追加のクレジットがかかります。`--lipsync` を追加してください。

### 音声分離とは何ですか？

ソースを音声／背景音／サブ背景音のトラックに分割します — 吹き替えは行いません。`--separate` を追加してください。

### 吹き替えの代わりに字幕（SRT）を作成できますか？

はい — このパッケージには **`/srt`** スキルも同梱されています。Persoの音声認識（speech-to-text）を使って動画/音声/URLから元言語の字幕を抽出し、続けてエージェントが指定した言語に翻訳します（元のファイルと同じ場所に `<name>_<lang>_Subtitle.srt` として保存されます）。文字起こしだけが欲しい場合は、そう伝えれば `--transcribe-only` で実行され、翻訳は行われません。字幕抽出は、メディアの長さに応じて（言語ごとに）クレジットを消費します。

### 途中で停止しました（クレジット切れ、クラッシュ、シェルの強制終了）。どうすればいいですか？

実行中は進捗が状態ファイルに保存され続けます（`/dubbing` の場合は `*.dubresume.json`、`/srt` の場合は `*.srtresume.json`）。表示された `--resume "<state-file>"` コマンドを再実行すると、残りの部分だけを完了できます — 完了済みの部分はスキップされ、再課金されることはありません。

### クレジットが不足しました。どうやってチャージすればいいですか？

このスキルは Stripe の支払いリンクを生成できます（プランに応じて、サブスクリプション登録・プラン変更・クレジット購入のいずれか）。リンクはご自身で開いて支払ってください — エージェントが代わりに支払うことはありません。チャージ後は、表示された `--resume` コマンドで再開してください。

### ローカルファイルを保存せずに吹き替えできますか？

はい、単一（分割されていない）動画であれば可能です: `--no-save` を追加してください。結果はPersoワークスペース内に残り、ダウンロードされません。分割された動画は、結合後のファイルをローカルにダウンロードする必要があるため、通常どおり保存されます。

### `node` が見つからないと表示されます — どうすればいいですか？

このスキルには Node.js 18以上 が必要です。`node -v` で確認してください。入っていない場合は <https://nodejs.org> からLTS版をインストールするか、エージェントにインストールを依頼してから、再試行してください。

### スキルはどうやって更新しますか？

`npx perso-dubbing@latest` を実行するか、Claude Codeプラグインでは `/plugin update perso-dubbing` を実行してください。

### スキルはどのようなデータを収集しますか？

匿名の利用イベントのみです — 実行されたアクション、成功したかどうか、概算のカウント数、アプリバージョン、OSなど — インストールごとのランダムなIDでタグ付けされます。APIキー、ファイル名やメディアの内容、アカウント／メールアドレス、ワークスペースIDは一切含まれません。
