關於 `/dubbing` 技能的常見問題。安裝與使用方式請參考 [README](README.md)。

### 我需要準備什麼才能使用？

Node.js 18+ 以及一組 Perso Dubbing API 金鑰。安裝技能後，只要說「幫我配音這個影片」即可。→ [取得 API 金鑰](https://developers.perso.ai/api-keys)

### 如何註冊我的 API 金鑰？

第一次執行時會自動開啟金鑰檔案——請在該檔案中**僅貼上你的 API 金鑰**並儲存（會自動加密，且檔案隨後會被刪除）。**絕對不要將金鑰貼到聊天視窗中。** 手動檢查方式：`npm run key:check`。

### 使用需要付費嗎？

此技能的程式碼是免費的（MIT 授權），但配音處理是透過 Perso API 執行，會依點數計費：配音約為每秒 1 點數，對嘴約為 ×2，音訊分離約為 ×0.5。在 pro／business／enterprise 方案中，4K 來源影片會以 ×3 計費。實際計費以伺服器端結果為準。

### 可以輸入哪些內容？

本機檔案、整個資料夾（批次處理），或網址——包括 YouTube、TikTok、Google Drive 及 Vimeo。過大或過長的影片會自動分割、處理後再合併回來。

### 可以同時配音成多種語言，或一次處理多個檔案嗎？

可以。在同一個指令中放入多種語言（`--target en,ja,zh`）——來源檔案只會上傳並分割一次，之後每種語言重複使用。你也可以在同一次執行中混合多個檔案、資料夾與網址。

### 我的結果會儲存在哪裡？

預設會儲存在原始影片旁邊，或儲存在你用 `--out` 指定的資料夾中。每次執行也會在你的 Perso 入口網站（<https://portal.perso.ai>）中建立一個專案，你可以在那裡重新下載或取得其他格式。

### 什麼是對嘴（lip-sync）？

它會將嘴型動作對齊配音後的音訊。這項處理會在配音完成後執行，僅適用於影片，耗時明顯較長，且需要額外點數。加上 `--lipsync` 即可使用。

### 什麼是音訊分離？

它會將來源分離成人聲／背景／次背景音軌，不涉及配音處理。加上 `--separate` 即可使用。

### 執行到一半中止了（點數用盡、當機，或殼層被終止）。現在該怎麼辦？

執行過程中，進度會持續儲存至 `*.dubresume.json` 狀態檔案中。重新執行畫面印出的 `--resume "<state-file>"` 指令，即可僅完成剩餘的部分——已完成的部分會被略過，且不會重複計費。

### 我的點數用完了，該如何加值？

此技能可以產生 Stripe 付款連結（依你的方案，可訂閱、變更方案或購買點數）。你需要自行開啟連結並付款——代理程式不會代替你付款。加值完成後，使用畫面印出的 `--resume` 指令繼續執行即可。

### 可以不儲存本機檔案就完成配音嗎？

可以，僅限單一（未分割）的影片：加上 `--no-save`。結果會保留在你的 Perso 工作區中，不會下載。分割過的影片仍會照常儲存，因為合併後的檔案需要下載到本機。

### 找不到 `node`，該怎麼辦？

此技能需要 Node.js 18+。可用 `node -v` 確認版本；請至 <https://nodejs.org> 下載 LTS 版本，或直接請代理程式幫你安裝，再重試一次。

### 如何更新此技能？

執行 `npx perso-dubbing@latest`，或在 Claude Code 外掛程式中執行：`/plugin update perso-dubbing`。

### 此技能會收集哪些資料？

僅收集匿名使用事件——執行了哪個動作、是否成功、粗略的計數、應用程式版本以及作業系統——並以隨機產生的每次安裝專屬 ID 標記。絕不包含你的 API 金鑰、檔案名稱或媒體內容、帳號／電子郵件，或工作區 ID。
