Claude Code(Claude 데스크톱 앱)에 Perso Dubbing 스킬을 설치하면, 영상 여러 개를 여러 언어로 한 번에 더빙할 수 있습니다. 업로드부터 파일 정리까지 대화로 끝납니다. Codex · Cursor · Antigravity에서도 그대로 동작합니다.

지금 쓰시는 Perso 계정과 크레딧을 그대로 사용합니다.

💡에이전트를 여러 개 쓰신다면 npx perso-dubbing 한 번이면 됩니다 — 설치된 호스트를 자동으로 감지해 전부에 설치합니다. 스킬 실행에는 Node.js 18 이상이 필요합니다.

더빙 요금은 지금 워크스페이스에서 쓰시는 것과 동일하게 보유 크레딧 한도 내에서, 처리한 분량만큼 차감됩니다. 막히는 부분은 FAQ에 정리돼 있습니다.

이런 작업이 자동이 됩니다

  • 폴더째, 유튜브 URL도 — 파일·폴더·링크를 섞어 맡겨도 됩니다
  • 긴 영상 자동 분할 · 병합 — 플랜의 월 처리 한도 내에서
  • 여러 언어를 한 번에 — 업로드는 1회, 언어별로 재사용
  • 립싱크 · 음성/배경음 분리 — 문장으로 요청하면 끝
  • 중단돼도 이어서 — 저장된 지점부터 재개
  • 결과는 원본 옆에 — 언어별로 정리되어 저장

준비물은 두 가지입니다

  • Claude 유료 플랜 — 데스크톱 앱·CLI의 Claude Code에서 실행합니다 (Codex · Cursor 등 다른 에이전트는 해당 없음)
  • Perso API 키 — 지금 로그인한 계정으로 바로 발급됩니다

Code 탭에서는 Local 세션을 선택하세요. Windows는 Git · Node 18 이상이 필요합니다 — 없으면 Claude에게 설치를 부탁하면 됩니다.
