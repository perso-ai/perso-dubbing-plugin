Claude Code(Claude 데스크톱 앱)에 Perso Dubbing 스킬을 설치하면, 영상 여러 개를 여러 언어로 한 번에 더빙할 수 있습니다. 업로드부터 파일 정리까지 대화로 끝납니다. Codex · Cursor · Antigravity에서도 그대로 동작합니다.

지금 쓰시는 Perso 계정과 크레딧을 그대로 사용합니다.

설치 방법 보기 API 키 발급
claude — videos/ — 80×24
$ claude> videos 폴더 전부 영어, 일본어, 스페인어로 더빙해줘.영상 31개를 찾았습니다. 3개 언어로 더빙을 시작할게요 — 긴 영상은 자동으로 분할해서 처리합니다.✔ promo_launch.mp4 → en·ja·es 완료✔ tutorial_02.mp4 → en·ja·es 완료⠧ interview_full.mp4 (47분 · 자동 분할 4/6) 68%○ shorts_015.mp4 외 27개 대기결과는 각 원본 옆 폴더에 저장됩니다

쓰시는 에이전트에 맞게 설치하세요

모든 호스트가 같은 Agent Skills 표준(SKILL.md)을 쓰기 때문에, 어디에 설치해도 /dubbing 하나로 똑같이 동작합니다.

1
Code 탭을 여세요 — Claude 데스크톱 앱(유료 플랜) 상단 중앙의 Code 탭에서 아무 폴더나 열고, 환경은 Local을 선택합니다. 클라우드 세션에서는 플러그인이 동작하지 않습니다.
2
두 명령을 한 줄씩 채팅창에 입력하고 Enter를 누르세요.
claude marketplace add perso-ai/perso-dubbing-plugin
claude install perso-dubbing@perso-ai
3
더빙을 요청하세요 — 예: "이 영상 영어로 더빙해줘 — C:\videos\clip.mp4" (유튜브 URL이나 폴더 전체도 됩니다). 첫 실행 때 API 키를 넣을 파일이 자동으로 열립니다 — 채팅이 아니라 그 파일에 키를 붙여넣으세요.
1
터미널에서 Claude Code를 실행하세요 — 작업할 폴더로 이동한 뒤 시작합니다.
claude
2
두 명령을 한 줄씩 입력하세요.
/plugin marketplace add perso-ai/perso-dubbing-plugin
/plugin install perso-dubbing@perso-ai
3
더빙을 요청하세요 — /dubbing을 입력하거나 "이 폴더 영상 전부 영어로 더빙해줘" 라고 말하면 됩니다. 첫 실행 때 API 키를 넣을 파일이 자동으로 열립니다.
1
터미널에서 한 줄 — 패키지를 받아 Codex 스킬 폴더에 설치합니다.
npx perso-dubbing --codex

설치 위치: ~/.agents/skills/dubbing/ (구버전 Codex용 ~/.codex/skills/에도 함께 기록됩니다)

2
Codex에서 바로 사용 — /dubbing을 입력하거나 "이 영상 더빙해줘" 라고 말하면 됩니다. 첫 실행 때 API 키 파일이 자동으로 열립니다.
1
터미널에서 한 줄 — 패키지를 받아 Cursor 스킬 폴더에 설치합니다.
npx perso-dubbing --cursor

설치 위치: ~/.cursor/skills/dubbing/ · 현재 프로젝트에만 설치하려면 npx perso-dubbing --cursor --project

2
Cursor에서 바로 사용 — /dubbing을 입력하거나 "이 영상 더빙해줘" 라고 말하면 됩니다. 첫 실행 때 API 키 파일이 자동으로 열립니다.
1
터미널에서 한 줄 — 패키지를 받아 Antigravity 스킬 폴더에 설치합니다.
npx perso-dubbing --antigravity

설치 위치: ~/.antigravity/skills/dubbing/ (1.x) · ~/.agents/skills/dubbing/ (2.0+)

2
Antigravity에서 바로 사용 — /dubbing을 입력하거나 "이 영상 더빙해줘" 라고 말하면 됩니다. 첫 실행 때 API 키 파일이 자동으로 열립니다.
💡에이전트를 여러 개 쓰신다면 npx perso-dubbing 한 번이면 됩니다 — 설치된 호스트를 자동으로 감지해 전부에 설치합니다. 스킬 실행에는 Node.js 18 이상이 필요합니다.

더빙 요금은 지금 워크스페이스에서 쓰시는 것과 동일하게 보유 크레딧 한도 내에서, 처리한 분량만큼 차감됩니다. 막히는 부분은 FAQ에 정리돼 있습니다.
