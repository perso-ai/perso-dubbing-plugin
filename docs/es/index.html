폴더째 배치

웹에서는 영상 1개씩. 에이전트는 폴더째, 긴 영상은 알아서 나눠서.

파일·폴더·유튜브 링크를 섞어 맡겨도 알아서 정리해 처리하고, 여러 언어도 한 번에 끝냅니다. 개당 길이 제한 없이 자동 분할·병합 — 플랜의 월 처리 한도 내에서.

에이전트 전용 — 웹에는 없어요

번역 자막만 필요한데, 더빙 요금을 내고 계셨나요?

"자막만 만들어줘"라고 하면 STT 비용만 차감됩니다. 번역은 대화 중인 AI가 그 자리에서 무료로 — 웹 서비스에는 없는, 에이전트로만 쓸 수 있는 기능입니다.

걸어두면 끝

퇴근 전에 걸어두세요. 아침엔 원본 옆에 다국어 영상이 정리돼 있습니다.

큐가 차면 알아서 기다렸다 다시 시도하고, 중간에 끊겨도 이중 과금 없이 저장 지점부터 이어갑니다. 결과는 내 컴퓨터 원본 옆에 언어별로 저장 — 다운로드 버튼 누를 일이 없습니다.
